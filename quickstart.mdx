Tododeia spawns 5 specialized AI agents to research crypto, stocks, forex, and commodities in parallel, then serves the results as an interactive dashboard at `http://localhost:3420`. This guide shows the fastest path from install to report.

<Warning>
  Tododeia is for informational and educational purposes only. It does not constitute financial advice. Always consult a qualified financial advisor before making investment decisions.
</Warning>

## Before you start

Make sure you have:
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed
- Node.js 18+ (for the interactive dashboard)
- An active internet connection (agents perform live web research)

## Run your first analysis

<Steps>
  <Step title="Install Tododeia">
    Run the one-liner installer in your terminal:

    ```bash
    curl -sL https://raw.githubusercontent.com/Hainrixz/maia-skill/main/install.sh | bash
    ```

    The script clones the repository, symlinks the skill into Claude Code, and installs dashboard dependencies automatically.
  </Step>

  <Step title="Open Claude Code and trigger an analysis">
    In any Claude Code conversation, use one of these trigger phrases:

    - `Run an investment analysis`
    - `What are the best investment opportunities today?`
    - `Analyze the markets`
    - `Give me a market report`
    - `Run tododeia`

    The skill activates automatically when you mention investment-related topics.
  </Step>

  <Step title="Choose your risk profile">
    Claude will ask for your risk tolerance before any research begins. Pick the profile that matches your goals:

    | Profile | Description |
    |---|---|
    | **Conservative** | Capital preservation, stable returns, lower risk (bonds, blue chips, gold) |
    | **Moderate** | Balanced growth and safety, diversified across sectors *(Recommended)* |
    | **Aggressive** | Maximum growth potential, comfortable with high volatility (crypto, growth stocks, leveraged positions) |

    Your selection shapes every recommendation: which assets are emphasized, position sizes, and portfolio allocation percentages.
  </Step>

  <Step title="Wait for the 5 agents to finish">
    Once you confirm your risk profile, Tododeia launches all agents:

    - **4 sector agents** run in parallel, each researching crypto, stocks, forex, and commodities using live web data
    - **1 strategy agent** synthesizes all sector findings into a cross-sector analysis, risk-adjusted rankings, and portfolio allocation

    This typically takes 1–3 minutes depending on your connection speed.
  </Step>

  <Step title="View your dashboard">
    When the analysis is complete, open your browser and navigate to:

    ```
    http://localhost:3420
    ```

    You'll see:
    - An **executive summary** with macro environment assessment
    - A **portfolio allocation** chart with recommended percentages per sector
    - **Risk-adjusted top picks** with confidence scores and suggested position sizes
    - **Cross-sector insights** highlighting correlations between markets
    - **Expandable sector panels** with detailed per-asset analysis
    - **Historical accuracy** tracking past recommendation performance

    Use the language toggle in the **toolbar** to switch between English and Spanish at any time.
  </Step>
</Steps>

## Set up recurring reports (optional)

After your first report, you can schedule automatic analysis:

```
/loop 24h /investment-analysis    # Daily reports
/loop 168h /investment-analysis   # Weekly reports
```

<Tip>
  Tododeia never caches data. Every invocation runs fresh web research across all sectors.
</Tip>
