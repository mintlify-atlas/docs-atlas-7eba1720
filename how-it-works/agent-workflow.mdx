Tododeia uses 5 specialized AI agents: 4 sector researchers that run in parallel, and 1 strategy agent that synthesizes their combined output. Each agent has a distinct role, a defined output schema, and clear error-handling rules.

## Parallelism and sequencing

The 4 sector agents launch simultaneously in a single orchestrator message. This means research across crypto, stocks, currencies, and commodities happens concurrently — the total research time is bounded by the slowest agent, not the sum of all four.

The Strategy Agent launches only after all 4 sector agents have returned their results. It receives every sector report at once and performs cross-sector analysis that no individual sector agent can do on its own.

```text
Step 4: ┌─────────┐ ┌─────────┐ ┌─────────────┐ ┌──────────┐
        │ Crypto  │ │ Stocks  │ │  Currencies │ │Materials │  ← parallel
        └────┬────┘ └────┬────┘ └──────┬──────┘ └────┬─────┘
             │           │             │              │
             └───────────┴─────────────┴──────────────┘
                                   │
                                   ▼
Step 5:                   ┌─────────────────┐
                          │ Strategy Agent  │  ← sequential
                          └─────────────────┘
```

<Note>
  Each sector agent must verify prices from at least 2 independent sources before reporting. `source_agreement` reflects how closely those sources agree: `"high"` means within 1%, `"medium"` within 1–3%, and `"low"` means sources diverge by more than 3%.
</Note>

## The Strategy Agent

The Strategy Agent acts as the Chief Investment Strategist. It receives all 4 sector reports, your risk profile, and any historical data from prior runs. It does not re-research prices — its value is in synthesis.

Its analysis framework covers five steps:

1. **Macro environment assessment** — reads interest rate direction, inflation signals, risk appetite, and geopolitical risk across all 4 sector reports.
2. **Cross-sector correlation analysis** — identifies meaningful patterns (e.g., gold and crypto both rising suggests fiat devaluation hedging; oil up with stocks down signals stagflation risk).
3. **Risk-adjusted ranking** — scores every asset with volatility adjustments that depend on your risk profile, then ranks them.
4. **Portfolio allocation** — distributes a hypothetical portfolio across sectors and cash, totaling 100%.
5. **Historical accuracy check** — if prior reports exist, compares previous buy/sell calls to current prices and calculates a directional accuracy percentage.

## Error handling

| Situation | Behavior |
|-----------|----------|
| `WebSearch` returns no results for an asset | Retry with `WebFetch` on Yahoo Finance, CoinGecko, or Google Finance |
| Sector agent returns malformed JSON | Re-prompt once with correction instructions; if it fails again, mark the sector as `"data_unavailable": true` |
| Strategy Agent fails entirely | Fall back to simple confidence-score ranking and note "Strategy analysis unavailable" in the report |
| Python not available to serve fallback HTML | Try `npx serve output -p PORT` or instruct you to open `output/report.html` directly |
| All web searches fail | Generate the report with "No data available" messages rather than crashing |
| Historical data files are corrupted | Skip accuracy tracking and start fresh |

<Tip>
  The `data_unavailable` flag on `SectorData` is surfaced in the dashboard so you can tell at a glance if a sector's data is missing rather than seeing empty or stale numbers.
</Tip>
